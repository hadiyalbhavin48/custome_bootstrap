Venquis Website

About Venquis

Venquis is a recruitment company that focuses on global businesses that want to scale.

About the project

Venquis realised that their brand no longer reflects their business anymore and invested in a rebranding. They hired a visual designer to work on the brand identity, a copywriter to work on the tonality and me to work on the website redesign.

The Goal

The aim of this project was to generate more leads from prospect clients and reflect Venquis' new brand identity.

The Target Audience

The Venquis website was mainly targeted at prospect clients while considering potential employees both for Venquis and their clients.

Sitemap and wireframes

Breite Font Weights

Challenge

One of the biggest challenges about the UI was probably the font type chosen. Breite is a very appealing and clean font, however, it doesn't come in a bold font weight, so the only way to play with the font hierarchy was to use different font sizes or upper case. Colour was an option too but the client was keen on keeping the font 100% black, even after I pointed out that 100% black on digital devices can cause eye straining to the user.

No error message shows when the user forgets to select a size. Instead, the size picker appears.

Accessibility

When visual appeal is in the forefront of a brand - which is often the case with fashion brands - accessibility can often take a hit.

Due to the challenge of the limited Breite font weights and the client not wanting to use another colour than black, it was difficult to show different hierarchy in some parts of the website.

The client wanted to use the 'thin' font weight to show a different hierarchy between product title and paragraph. Even as someone with somewhat of a good eyesight, it was difficult to read, so for users with impairments like dyslexia, it would be way worse. As much as it was nice to look at, it wasnâ€™t ideal to read.

Luckily, one of the brand's values is to be accessible for everyone, so I could convince the client to allow the font colour to be grey instead of using a 'thin' font version of Breite.

Learnings

I have learnt a lot from the Visual Designer and vice versa. It was great to work with someone who knows the brand inside out and was able to ensure the branding is represented across each page without losing impact due to other brands being sold on the market place.

This project was also my first e-commerce project and dipping into a new type of product is always a welcomed challenge.
