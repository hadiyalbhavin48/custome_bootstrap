AI Software Insight

About

InSight is an AI platform that analyses data within seconds, it clusters the data and provides useful insights. This project was a redesign of an existing software.

Target Audience

The target audience for this product are data analysts. After conducting user interviews it became clear that the users wanted the ability to control the amount of clusters and review the individual data points within a cluster, despite the automation. They also wanted to be able to view results in a different approach which should allow them to find more insights.

User Interface

Jaggu is an AI start up that provides several software solutions. To ensure all software is recognisable, I created a UI Kit to be used across all Jaggu software to ensure consistency and brand recognition. This software pulled from the same UI Kit.
