Croppd - AI powered Image Cropping Tool

About Croppd

Croppd is an AI software that automatically crops an uploaded image to set sizes. This tool was requested by a marketing client which manually cropped images for online banner ads.

User Interface

This software, just like other software I designed during my time as a Head of Design at Jaggu, pulled from the same UI Kit I designed to ensure consistency and recognisability across all Jaggu tools.

Simple Elements

Croppd is a very simple and straight forward software with the only purpose to crop one uploaded image to different sizes. To make this automated process quick and easy, the few areas where the user needs to interact needed to be as simple as possible.

Clear Visibility when Action is required

When uploading images, the AI will automatically crop the images to the selected sizes. When the system notices potential issues, the user sees a clear status when action is required.



Real-size Preview

The user can review all images in the final sizing. In case the images were edited by the AI, hovering over the image will reveal it fully. The user can then confirm or decline the image.



Machine Learning on each manual input

After the user has manually approved or rejected an automatically cropped image, a visual cue is shown.
