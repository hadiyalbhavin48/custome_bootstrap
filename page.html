FAQs

UI design, or user interface design, is designing a digital product's visual and interactive elements, basically the visual part you see on a website or mobile app. This can include everything from layout and typography to buttons and icons, animations and interactions. See how I can help you with your UI web design here.
UI design is about designing a digital product's visual and interactive elements, while UX design is about creating the overall experience for the user with the product. In other words, UI focuses on how the product looks, while UX centres on how the product works and feels.
